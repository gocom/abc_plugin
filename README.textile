h1. abc_plugin

p. "Download":https://github.com/gocom/abc_plugin/archive/master.zip | "Packagist":https://packagist.org/packages/rah/abc_plugin

p. This is a plugin template for "MassPlugCompiler":https://github.com/gocom/MassPlugCompiler.

p. This would be your plugin's help file.

h2. Installing

p. This plugin template can be installed using "Composer":http://getcomposer.org:

bc. $ composer.phar require rah/abc_plugin

h2. Cloning

p. This plugin template can be cloned, and forked, using "git":http://git-scm.com/:

bc. $ git clone https://github.com/gocom/abc_plugin.git

h2. Compiling the plugin template

p. The template can be compiled to a plugin installer using "MassPlugCompiler":https://github.com/gocom/MassPlugCompiler:

bc. $ git clone https://github.com/gocom/MassPlugCompiler.git
$ compile /absolute/path/to/abc_plugin
