h1. abc_plugin

"Download":https://github.com/gocom/abc_plugin/archive/master.zip | "Packagist":https://packagist.org/packages/rah/abc_plugin

This is a plugin template for "Textpattern CMS":https://textpattern.com.

This would be your plugin's help file.

h2. Install

This plugin template can be installed using "Composer":https://getcomposer.org:

bc. $ composer require rah/abc_plugin

h2. Cloning

This plugin template can be cloned, and forked, using "git":https://git-scm.com/:

bc. $ git clone https://github.com/gocom/abc_plugin.git

h2. Compiling the plugin template

The template can be compiled to a plugin installer package using "mtxpc":https://github.com/gocom/MassPlugCompiler, which is included as a development dependency:

bc. $ git clone https://github.com/gocom/abc_plugin.git && cd abc_plugin
$ composer install
$ composer compile
